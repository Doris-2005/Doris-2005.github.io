Personal Project

Canvas Assignment Reminder

A rule-based automation tool that converts Canvas iCal events into organized Apple Reminders tasks.

Overview

This tool reads iCal calendar data provided by Canvas, extracts course names, task titles, due dates, and item types, then filters important academic tasks such as assignments, exams, and reports based on keywords and rules. After processing, the system exports the results into a format readable by Apple Shortcuts, allowing the tasks to be imported into Apple Reminders on iPhone or iPad.

To avoid duplicate reminders, the tool records previously processed items and only imports new tasks each time it runs.

Workflow

  1. Read Canvas iCal data
  2. Parse calendar events
  3. Extract course name, task title, and due date
  4. Classify tasks as Assignment, Exam, or Report
  5. Remove already imported items
  6. Export data for Apple Shortcuts
  7. Import tasks into Apple Reminders
  8. Check and complete tasks in Reminders

Design Focus

The motivation for this tool came from the repeated need to check assignment, exam, and report deadlines on Canvas and manually enter them into a reminder system. To reduce repetitive work and avoid missing deadlines, this project automates the process of extracting important academic tasks and organizing them into Apple Reminders.

The tool works as a rule-based agent. It analyzes task names, course information, and keyword rules to determine whether each item belongs to Assignment, Exam, or Report. It also includes a record of processed items to prevent the same task from being imported repeatedly.

Key Features

  • Reads Canvas iCal calendar data
  • Extracts course names, task titles, and due dates
  • Classifies tasks into Assignment, Exam, and Report
  • Filters out duplicate imported items
  • Exports data for Apple Shortcuts
  • Imports tasks into Apple Reminders
  • Helps manage academic deadlines more clearly
