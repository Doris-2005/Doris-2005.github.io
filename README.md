# Lin Tzu-Ting Portfolio

A minimal personal portfolio website built with pure HTML, CSS, and JavaScript. It can be deployed directly to GitHub Pages without any build tools.

## Files

- `index.html`
- `style.css`
- `script.js`
- `assets/`

## GitHub Pages Deployment

1. Create a repository named `username.github.io`.
2. Upload all files in this project to the repository.
3. Go to `Settings` -> `Pages`.
4. Set `Source` to the `main` branch.
5. Open `https://username.github.io/`.

## Customization

Update the placeholder text, project links, email address, and GitHub URL in `index.html`.
